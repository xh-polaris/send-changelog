# send changelog action

Action used to send the changelog to the webhook of lark.

## Inputs

### `webhook_url`

**Required** The webhook of robot.

## Example usage

```yaml
uses: xh-polaris/send-changelog@v1.0.0
with:
  webhook_url: https://example.com/path
```
